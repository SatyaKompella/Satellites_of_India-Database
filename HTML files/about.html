About

Satellite databases contain all the data about the present and past satellites that orbit Earth. In our database, we aim to do the same, and have collected data from INSAT and ISRO to populate it. With the number of satellites growing every year, there is a need for a proper database that contains all the details. We ensure efficient user interface by providing flexible fields for users to enter data. With time, we hope to convert it into a real time database that has the ability to track satellites.
